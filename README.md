# Nuxt-blog

New blog website using Nuxt deployed to github pages. Uses Nuxt to enable creating interactive blogs using Nuxt Content.

## Tech Stack

- Nuxt
    - Nuxt UI
    - Nuxt Content
- GitHub Pages
